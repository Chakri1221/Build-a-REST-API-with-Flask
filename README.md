# Bookstore REST API using Flask & SQLAlchemy
This project is a RESTful API developed using Flask and SQLAlchemy for managing bookstore data. The API allows users to perform CRUD (Create, Read, Update, Delete) operations on books stored in an SQLite database.

The system includes features such as adding new books, viewing all books, searching books by title or author, updating book details, and deleting books from the database. The project follows REST API principles and returns data in JSON format for easy integration with frontend applications or external services.

For database management, SQLite is used along with SQLAlchemy ORM to simplify database operations. The API endpoints were tested using Postman to verify request handling, responses, and CRUD functionality.

🔹 Key Features
Add new books to the database
View all books
Search books by title or author
Retrieve a single book by ID
Update existing book details
Delete books from the database
JSON-based REST API responses
SQLite database integration
API testing using Postman
